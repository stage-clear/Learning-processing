- __運動のアルゴリズム__ :
 - 新しい位置 = 現在の位置に速度を適用
 - 位置(location) = 位置 + 速度(velocity)
- __ベクトルの長さ__ :
 - ピタゴラスの定理を使って斜辺の長さを割り出します
 - `‖v→‖ = √vx * vx + vy * vy`
- __ベクトルの正規化__ :
 - ベクトルの長さで, 各辺を割ります
 - `u^ = u→ / ‖u→‖`
